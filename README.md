# Los Jojaras – GitHub Pages site

## Så publicerar du

1. Skapa ett nytt repository på GitHub, till exempel `los-jojaras`.
2. Ladda upp alla filer i den här mappen till repositoryts root.
3. Gå till **Settings → Pages**.
4. Under **Build and deployment**, välj:
   - Source: **Deploy from a branch**
   - Branch: **main**
   - Folder: **/root**
5. Spara. Efter någon minut finns sidan på GitHub Pages-adressen.

## Ändra innehåll

- Text och länkar ändras i `index.html`.
- Färger, layout och mobilanpassning ändras i `styles.css`.
- Bilder ligger i `assets/images/`.
